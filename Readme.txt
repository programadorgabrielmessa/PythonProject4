Este é um script que uso para vasculhar links Wordpress para saber os links de como abrir os sites.
Fez esse script com interface porque me encontrava perdido pra fazer meus novos cursos complementares e não vi solução nenhuma na internet.
Foi então que criei este pequeno aplicativo que vasculha em todo os discos rígidos e mostra as pastas e links que podem ser abertos.
Talvez seja até de muita utilidade até pra quem é freelancer e se encontra perdido em seus sites ou esqueceu alguma coisa.
